Algumas tags estão obsoletas!

No HTML4 era normal usar tags para da forma ao front, mas isso mudou com a chegada do CSS e agora é errado usar tags de formatos no HTML.

Atualemnte o HTML é focado apenas na semântica, tags com significado! Por Exemplo:

Rua Adalia 768, Bairro Florida , Contagem-MG

Para inserir endereços é utilizado a tag <address>, ela já exibe na tela em um formato diferente apenas para endereço.
