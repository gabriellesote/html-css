Principais formatações

Negrito / Destaque

Para fazer o Negrito, é usado a tag <strong> pois é uma tag semântica, melhor usá-la no HTML.
Diferente da tag <b> que devse ser usada no CSS por ser uma tag de formato!

Itálico / Ênfase

Para fazer o Itálico, é usado a tag <em> pois é uma tag semântica. Diferente da tag <i> que deve ser utilizada no CSS por ser uma tag de formato!

Texto marcado

Para fazer m texto marcado, é usado a tag <mark>. Aparentemente o VSCODE ainda não automatizou a tag mark.
Caso queira que todos os marks sejam iguais, faça: 

            
                <head>
                  <style>
                  mark{
                    background-color: cor;
                  }
                  </style>
                </head>

Texto grande e pequeno

Para por o texto grande era usado a tag <big>, porém, esta tag está obsoleta pois não há sentido em utilizar uma tag HTML sendo que tem o CSS para a formatação.

Para por o texto pequeno ainda é usado a tag <small>.

Texto Deletado

Tag: <del> 'deleta' o texto
Tag: <ins> sublinha o texto

Texto Sobrecrito e Subscrito

Tag: <sup> 242 deixa o texto 'elevado'
Tag: <sub> H2O deixa o texto 'abaixado'
